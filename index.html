Selected Projects

Numerical Precision Analysis

This project investigates the limitations of hardware-accelerated data types by analyzing floating-point precision decay and summation errors. Utilizing Python and the Matplotlib library, the study visualizes how rounding errors accumulate during large-scale numerical computations, specifically comparing the accuracy retention of float32 versus float64 types. By calculating relative error metrics and plotting precision loss over time, the project provides a technical deep dive into the computational trade-offs between processing speed and mathematical rigor, serving as a critical resource for ensuring reliability in scientific computing and algorithmic development.
